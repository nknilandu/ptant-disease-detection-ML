About This Project

A visually improved plant disease detection web application using Flask, Bootstrap and a trained deep learning model.

Plant Disease Classification Using Deep Learning

This system is designed to help identify plant leaf diseases from images using a deep learning model. Users can upload a leaf image, preview it, and get an instant prediction result.

The application is useful as an academic project, prototype solution, or demonstration system for agriculture-related AI applications. The goal is to make plant health monitoring simpler and faster.

Future upgrades can include confidence score, treatment suggestions, disease details, prevention tips and multi-language support.
